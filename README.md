# homebrew-tap

Homebrew tap for BRO3886 CLI tools.

## Install

```bash
brew tap BRO3886/tap
```

Then install any of:

```bash
brew install rem        # macOS Reminders CLI
brew install ical       # macOS Calendar CLI
brew install gtasks     # Google Tasks CLI
brew install healthsync # Apple Health export → SQLite
```

## Tools

| Formula | Description | Repo |
|---------|-------------|------|
| `rem` | Blazing fast CLI for macOS Reminders | [BRO3886/rem](https://github.com/BRO3886/rem) |
| `ical` | Fast, native macOS Calendar CLI | [BRO3886/ical](https://github.com/BRO3886/ical) |
| `gtasks` | CLI tool for Google Tasks | [BRO3886/gtasks](https://github.com/BRO3886/gtasks) |
| `healthsync` | Apple Health export → SQLite | [BRO3886/healthsync](https://github.com/BRO3886/healthsync) |
